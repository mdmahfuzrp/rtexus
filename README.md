## Installation for First Time
1. Clone this repository:
```sh
git clone https://github.com/mdmahfuzrp/rtexus-frontend-task.git
```

## Install Package
3. Install related package
```sh
npm install or npm i
```

4. Run this command to see live site

```sh
npm run dev
```
